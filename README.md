## What is Tally Assistant?

Tally Assistant is an AI-native bookkeeping tool built for freelancers and independent contractors. It reads any bank CSV, scans payment screenshots from any platform, generates invoices from plain English, and automatically chases late payments — all with AI.

**The philosophy:** throw anything at it, and the AI figures it out. No accounting knowledge needed, no manual data cleanup, no rigid formats.

---

## Core Features

| Feature | Description |
|---------|------------|
| 📎 **Universal CSV Import** | Upload any bank CSV from 200+ banks and payment platforms — AI auto-detects columns, date formats, delimiters, and currencies in any language |
| 📸 **AI Screenshot Recognition** | Snap any payment screenshot (PayPal, Venmo, Alipay, WeChat) — AI extracts amount, sender, date, and currency |
| 🧾 **AI Invoice Generator** | Describe the work in one sentence — "Logo design $300, landing $700 for Acme Corp" — AI builds the invoice with line items, tax, and PDF |
| 🔔 **Auto Payment Reminders** | AI sends polite reminders that escalate from friendly to firm over 21 days — fully automated |
| 🌍 **30+ Currencies** | Daily exchange rates, per-transaction currency detection, everything converted to your base currency |
| 💬 **AI Finance Assistant** | Ask "How much profit in May?" or "Which client pays me the most?" — AI answers from your data |
| ⏱ **Time Tracking** | Real-time timer or manual entry, convert tracked time to invoices with one click |
| 📄 **PDF Bank Statement Import** | Upload PDF statements and AI extracts every transaction automatically |

## Why I Built This

I'm Dominic, a full-stack developer from Chengdu, China 🐼. As a freelancer, I spent 7+ hours every month reformatting bank CSVs, digging through camera rolls for payment screenshots, manually creating invoices, and awkwardly chasing late payments.

Existing tools either required accounting knowledge (QuickBooks, Xero) or couldn't handle messy real-world data (different CSV formats, European commas, Chinese headers, multi-currency chaos).

So I built Tally Assistant — a tool that accepts whatever data you throw at it and figures out the rest with AI.
